Textzy API Integration

Simple browser-friendly API mode + tenant security + SMS/WhatsApp endpoints.

1) Browser-Simple Send API No Bearer / No CSRF

Use backend URL directly from browser or server. Authentication uses query credentials.

GET https://textzy-backend-production.up.railway.app/api/public/messages/send?recipient=91XXXXXXXXXX&msg=YOUR_MESSAGE&user=MONEYART&pswd=YOUR_PASSWORD&apikey=YOUR_API_KEY&tenantSlug=moneyart&channel=sms

Required Query Params

  • recipient
  • msg
  • user
  • pswd
  • apikey

Optional

  • tenantSlug (if omitted, fallback to user)
  • channel: sms or whatsapp
Always call backend domain, not frontend domain.

2) Security

  • HTTPS mandatory
  • API credentials validated
  • Optional IP whitelist (IP/CIDR)
  • Tenant routing enforced

3) Configure Credentials

Platform settings scope: api-integration

  • enabled=true
  • apiUsername
  • apiPassword
  • apiKey
  • ipWhitelist (optional)
